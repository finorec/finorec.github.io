📂 Importer la liste depuis Pronote
💡
Utilisez la version web de Pronote, elle fournit directement le fichier à importer.
Comment exporter depuis Pronote Web :
  1. Connectez-vous à Pronote version Web (pas le client lourd).
  2. Allez dans Mes données > CLASSES/ÉLÈVES > Liste des élèves.
  3. Choisissez votre groupe ou votre classe.
  4. En haut à droite, cliquez sur le bouton Copier la liste (format CSV).
  5. Pronote télécharge automatiquement un fichier — c'est ce fichier que vous importez ici.
