Cette application s'adresse aux professeurs dont les DST ont un nombre de points par question bien défini, déterminant la note finale. Elle vous permettra de gagner du temps de correction tout en faisant pour vous des statistiques détaillées et des fiches personnalisées pour vos élèves : ils pourront y voir leur positionnement dans la classe, le détail des points et dans les capacités attendues, lesquelles sont acquises, en cours d'acquisition et non acquises. Par ailleurs, MonSantorin vous facilite la distribution des fiches (PDF) à vos élèves en proposant quatre façons différentes de les donner pour couvrir toutes les situations.

L.B

1. Le barème et les capacités attendues

Le barème se crée dans ✏️ Éditer le barème. Deux approches sont possibles :

Approche 1 — Saisie manuelle
Ajoutez vos exercices, parties et questions directement dans l'éditeur. Pour chaque question, renseignez son intitulé, son barème et, si vous le souhaitez, ses capacités attendues (les compétences évaluées) ainsi qu'une réponse courte. Les capacités attendues apparaissent sous forme de petits badges cliquables ; appuyez sur Entrée pour en valider une nouvelle. Le champ s'allonge au clic et propose des suggestions par ordre alphabétique.

Approche 2 — Génération par IA puis audit
Cliquez sur ✨ Créer barème, capacités & réponses : un prompt est copié dans votre presse-papier. Collez-le dans votre IA favorite en joignant votre sujet (et éventuellement votre corrigé). L'IA renvoie une structure complète (exercices, questions, points, capacités, réponses). Copiez sa réponse, collez-la dans le champ qui apparaît dans MonSantorin, puis cliquez sur ✅ Valider et appliquer.
Vous avez déjà saisi vos exercices à la main ? Utilisez plutôt 🤖 Ajouter capacités & réponses à un barème existant : l'IA respecte vos points et complète uniquement les capacités et réponses.

Dans les deux cas, vous pouvez affiner le barème à tout moment depuis l'éditeur :

  • Bouton ✏️ Renommer les capacités : corrigez une faute ou supprimez massivement une capacité partout à la fois (corbeille 🗑️).
  • Interrupteur 💡 Réponses (ou Alt + R) : masquez ou affichez les réponses courtes à la volée.
  • Bilan automatique : une fois toutes les questions d'un élève notées, son bilan de capacités est généré et mis à jour automatiquement.
  • Code source dans les intitulés et réponses : vous pouvez inclure du code informatique dans les libellés de questions et les réponses.
    Saisie manuelle : accents graves pour le code en ligne (`variable`) ou bloc multiligne (```python … ```).
    Réponses générées par une IA : le prompt MonSantorin demande les balises dédiées [[python]]…[[/python]], [[sql]]…[[/sql]], etc. (sans accents graves, compatibles Gemini). MonSantorin les convertit automatiquement en affichage code.
    Langages pris en charge : Python, SQL, JavaScript, HTML… Police monospace et indentation préservées dans MonSantorin et les PDF.
2. Gestion des élèves
  • Import depuis Pronote : Cliquez sur 📂 Importer la liste Pronote des élèves. Une fenêtre vous guidera pas à pas. En résumé : dans la version web de Pronote, allez dans Mes données > CLASSES/ÉLÈVES > Liste des élèves, choisissez votre groupe, puis cliquez en haut à droite sur Copier la liste (format CSV). Importez ensuite le fichier téléchargé dans MonSantorin.
  • Ajout manuel : Utilisez le bouton + Nouvel élève situé à la fin de la liste des copies (à gauche) pour ajouter une copie vide.
  • Élève absent : Cliquez sur le bouton 🚫 ABS sur la fiche de l'élève pour le marquer comme absent. Il apparaît en grisé dans la liste. Un élève absent est exclu des statistiques de classe, ignoré lors de la génération du ZIP et de la communication des notes. Aucun PDF ne lui est généré. Cliquer à nouveau sur le bouton retire le statut absent.
  • Supprimer un élève : Utilisez le bouton 🗑️ sur la fiche de l'élève.
3. Notation

Utilisez les flèches ↑ ↓ pour passer d'une question à l'autre et les flèches ← → pour baisser ou augmenter la note — ou les menus déroulants si vous préférez la souris. Tapez les premières lettres du nom d'un élève ou cliquez sur son nom dans la liste pour accéder directement à sa page de notation.

4. Sauvegarde des notes

Une fois la correction terminée, cliquez sur le bouton jaune 💾 Sauvegarder Correction en bas. Cela génère un fichier contenant toutes les notes et appréciations, à conserver précieusement.

5. Fiche de notation individuelle

Cliquez sur le bouton noir 📄 PDF (Individuel) pour les élèves qui demandent le détail de leur note.

Vous pouvez également télécharger toutes les fiches d'un coup via le bouton 📦 Télécharger les PDF de la classe. Le fichier généré est une archive ZIP à décompresser avant de communiquer les fiches aux élèves.

💡 Astuces d'exportation : Depuis le bouton ⚙️ Paramètres (en haut de page), vous pouvez :
- Masquer le bloc "Appréciation" sur les PDF.
- Générer un PDF global de toute la classe.
- Activer une page blanche automatique pour le recto-verso (utile pour l'impression).
- Ajouter un Graphique de réussite individuel.
6. Distribution des copies (Nuage ou Monlycee.net)

MonSantorin propose quatre méthodes pour distribuer les fiches PDF aux élèves et à leurs familles :

  • Nuage (Dossier unique & Mots de passe) : un seul lien public pour toute la classe, chaque copie verrouillée par un mot de passe PDF personnel (script Python requis).
  • PDF chiffrés — diffusion manuelle : MonSantorin chiffre les copies dans le navigateur ; vous les diffusez vous-même (Nuage, Pronote, messagerie…). Aucun Python.
  • Nuage (Liens individuels) : un lien de téléchargement unique et anonyme par élève (script Python requis).
  • Monlycee.net : envoi direct par mail sur la messagerie monlycee.net (lycées Île-de-France uniquement), élèves et parents en option (script Python requis).

Choisissez votre méthode dans ⚙️ Paramètres (section « Mode de communication ») avant d'ouvrir la modale 📧 Communication. Les modes Nuage automatique et monlycee.net nécessitent Python sur votre ordinateur ; le mode « diffusion manuelle » fonctionne entièrement dans le navigateur.

A. Nuage Apps Éducation — Dossier public sécurisé par mots de passe

Vous ne donnez qu'un seul lien à toute la classe. Chaque PDF est chiffré avec un mot de passe personnel calculé à partir du nom de l'élève.

🛠️ Configuration (première utilisation uniquement) :
  • URL WebDAV : Dans votre Nuage, cliquez sur Paramètres de Fichiers (en bas à gauche) et copiez l'adresse WebDAV (du type https://nuageXX.apps.education.fr/remote.php/dav/files/votre.identifiant/). Renseignez-la dans ⚙️ Paramètres (section « Stockage WebDAV »).
  • Clé secrète de l'établissement : Renseignez-la dans ⚙️ Paramètres (section « Mots de passe PDF »). La même clé doit être utilisée par tous les professeurs de l'établissement.
  • Mot de passe d'application Nuage : Dans votre Nuage > icône profil > Paramètres > Sécurité > bas de page > créez un mot de passe d'application nommé "MonSantorin".

Homonymes : deux élèves portant exactement le même nom auront le même mot de passe PDF. Ce cas reste rare dans une même classe.

B. PDF chiffrés — diffusion manuelle (sans Python)

MonSantorin génère une archive ZIP de copies PDF, chacune verrouillée par le mot de passe personnel de l'élève (calculé à partir de son nom et de la clé établissement). Vous diffusez ensuite les fichiers et les mots de passe comme vous le souhaitez.

🛠️ Configuration (première utilisation uniquement) :
  • Clé secrète de l'établissement : Renseignez-la dans ⚙️ Paramètres (section « Mots de passe PDF »). La même clé doit être utilisée par tous les professeurs de l'établissement.
  • Mode de communication : Sélectionnez « PDF chiffrés — diffusion manuelle (sans Python) » dans ⚙️ Paramètres.

Homonymes : deux élèves portant exactement le même nom auront le même mot de passe PDF. Ce cas reste rare dans une même classe.

C. Nuage Apps Éducation — Liens de partage individuels

Chaque élève reçoit un lien de téléchargement unique et anonyme vers sa copie.

Configuration (première utilisation) : renseignez l'URL WebDAV et l'identifiant dans ⚙️ Paramètres (section « Stockage WebDAV »).
D. Messagerie monlycee.net — Envoi par mail direct

Réservé aux lycées d'Île-de-France. Les copies sont envoyées directement sur les adresses @monlycee.net des élèves (et de leurs parents en option).

🛠️ Configuration (première utilisation uniquement) :
  • Base ENT : Dans ⚙️ Paramètres (section « Base ENT monlycee.net »), importez le fichier CSV de la base élèves/parents. Ce fichier est à demander à votre référent numérique ou à votre chef d'établissement : sa génération nécessite un accès à l'outil d'administration de la base de données monlycee.net de l'établissement. Champs obligatoires : Type, Nom, Prénom, Identifiant, Parents.
  • Serveur SMTP : Dans ⚙️ Paramètres (section « Serveur SMTP »), le serveur smtps.monlycee.net et le port 465 sont préremplis. Renseignez votre adresse @monlycee.net.
  • Autorisation SMTP : Connectez-vous à la messagerie monlycee.net > votre nom > Paramètres > Configuration de mes accès > IMAP & SMTP > activez le curseur SMTP.
7. Aménagement Tiers-temps (TT)

MonSantorin permet de gérer facilement les copies des élèves bénéficiant d'un tiers-temps :

  • Activation : Cliquez sur le bouton ⏱️ TT sur la fiche de l'élève.
  • Calculs automatiques : Continuez de saisir vos notes brutes question par question. MonSantorin se charge de majorer automatiquement de +33 % le total de la copie à l'affichage.
  • Rendu PDF : La mention "(TT)" est automatiquement masquée sur le PDF de la copie de l'élève, et une mention "Note majorée au titre du tiers-temps" est discrètement ajoutée sous le tableau des scores.
8. Sauvegarde et Restauration (Important !)
  • Sauvegarder : Utilisez le bouton jaune 💾 Sauvegarder Correction pour vos notes et téléchargez le fichier JSON de votre barème.
  • Restaurer : Importez d'abord votre fichier Barème (JSON), puis utilisez le bouton bleu 📥 Importer Correction sur la page principale pour retrouver vos notes.
